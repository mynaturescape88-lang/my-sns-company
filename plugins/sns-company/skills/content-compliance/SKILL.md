# コンテンツ法令・コンプライアンスレビュー スキル v1.0

すべての公開コンテンツ（ブログ・Instagram・Threads・YouTube・note・Pinterest）を公開・レビューするときに発動する。
**目的**：pm/CLAUDE.md にルールを書いて毎回読む方式をやめ、公開審査タスクのときだけ本スキルを発動して全ルールを過不足なく適用する。

## 発動時に必ず行う（発動＝適用）

1. 下記の全チェック（法律違反チェック・著作権チェック・ステマ/事実確認/日本語品質/コンテンツ安全性）を実施する。
2. 1項目でもNGの場合は、修正完了まで公開処理を停止する。
3. 末尾の「公開前チェックリスト」を通してから公開／承認する。

> このスキルは公開コンテンツ共通の**法令・著作権・事実・日本語・安全性レビュー専用**。個別コンテンツ種別固有のレイアウト/構成チェックは各専用スキル（shop-article・wordpress-publishing・affiliate-writing・note-writing・instagram-caption・threads-post・event-calendar）にある。

## 法律違反チェック（全コンテンツ共通・必須）

すべての公開コンテンツ（ブログ・Instagram・Threads・YouTube・note・Pinterest）に対して以下を確認する。

### 景品表示法
- 「No.1」「日本一」「最高」等の最上級表現に根拠があるか（なければ削除）
- 比較広告は客観的な根拠があるか
- 過大な効果・利益を示唆する表現がないか

### 薬機法
- 植物・商品の健康効果・治療効果を示唆する表現がないか
  例：「この植物で病気が治る」「体に良い」等はNG

### ステマ規制（PR表示義務）
- アフィリエイトリンク・PR案件を含むすべての投稿媒体で `#PR` `#広告` `#アフィリエイト` 等の明示があるか
- Instagram・Threads に限らず、ブログ・note・YouTubeも対象

### チェック結果がNGの場合
修正完了まで公開処理を停止する。

## 著作権チェック（全コンテンツ共通・必須）

| 対象 | チェック内容 |
|---|---|
| 画像 | Pixabay・Pexels・自撮り・公式提供素材以外の無断使用がないか |
| テキスト | 他サイトからの無断転用・コピペがないか |
| BGM・音楽 | ライセンス（CC0・商用利用可）の確認がされているか |
| ショップ写真 | 公式サイト等からの無断転載がないか |
| 動画の全素材 | 映像・静止画・ロゴ・公式画面スクショ・BGM・効果音・音声/ナレーション音源・フォントのすべてについて、**選定時点で**出所とライセンスを**配布元の一次規約**で確認済みか（サービス名/ファイル名からの推測・別サービスの規約の流用は不可）。**固定資産（OP/ED）・使い回し・過去に承認済みの素材も対象＝免除は無い** |
| ライセンス証跡 | 証跡ファイル（`LICENSE_AND_SOURCES.txt` 等）が**素材と同じディレクトリに実在**するか（素材の移動・整理時は証跡も一緒に運ぶ） |

著作権違反が疑われる素材を発見した場合は、使用停止または出典確認をしてから公開する。
**合法だと確認できない素材は使用不可**＝この基準を満たさない成果物は完了・提示・公開しない。**「オーナーがOKした」「前回も使った」「再選定しないと決めた」は権利確認の代替にならない**（凍結は制作上の凍結であって権利の再検証を封じない）。

**判定の型＝入手経路で基準を分ける**（「確認できない」と「NGと記載されている」は別物・2026-07-20 オーナー指摘）
| 素材の入手経路 | 合格の条件 |
|---|---|
| 第三者から取得した素材（実写・画像・ロゴ・公式画面・音源等） | **許諾を積極的に確認できること**。出所不明・規約に辿り着けないものは不合格（従来どおり） |
| 正規に提供された実行環境の同梱物（OS同梱フォント等） | **当該利用を禁止する条項が存在しないことを一次規約で確認できれば可**。「明示的な許諾条項が無い」ことだけを理由に不合格としない |

- 後者を一律不合格にすると、**正規に購入した環境の標準機能を使えないという過剰判定**になり、既に公開済みの成果物へ遡って影響する。禁止条項の有無を一次規約で確かめるのが正しい検査。
- **出所が記録されていない素材は、画素・波形などの性質の測定で出所を代替しない**（2026-07-20 監査指摘）。例＝「**彩度0だから写真ではない**」は誤り＝**白黒写真・グレースケール化イラスト・白黒ロゴ・スキャン印刷物はいずれも彩度0**。しかも**第三者が作った同種の物も同じ値を示す**＝**画素は作者を証言しない**。**判定は保護対象要素の分解**（文言／図像／書体／音源／背景・配置）で行い、各要素が「自組織のもの」か「非保護」かを示すこと。性質の測定が言えるのは**物の種別の限定まで**（例＝合成レンダであって撮影物ではない）＝有効なので捨てず、位置づけを明記する。
- **いずれの場合も、確認結果と根拠（一次規約URL＋条項番号＋禁止条項の有無）を台帳§Kへ記録する**ことは必須＝記録の免除にはならない。確認しきれない項目は空欄にせず「調査中」等と明記して起票する。

## コンテンツ共通レビュー基準（ステマ・事実・日本語・安全性）

### ステルスマーケティング（必須）
- 楽天アフィリエイトリンク・PR案件を含む投稿に `#PR` `#広告` `#アフィリエイト` 等の明示があるか
- 報酬・提供品が発生している場合、その旨が読者に伝わる表現になっているか

### 事実確認（必須）
- ショップ情報（店名・住所・営業時間・価格）に誤りがないか
- 植物名・品種名が正確か（写真ファイル名に記載のない名称を断定していないか）
- 誇大表現・根拠のない断定がないか（「日本一」「必ず」「完全」等）

### 日本語品質（必須）
- 文法が正しいか（助詞の誤用・主語と述語のねじれ等）
- 句読点・改行の位置が自然か（読みやすい区切りになっているか）
- 絵文字・記号の使いすぎ・不自然な使い方がないか

### コンテンツ安全性（必須）
- 差別的・ヘイト表現がないか
- 暴力・有害な表現がないか
- 誤情報・詐欺的な表現がないか
