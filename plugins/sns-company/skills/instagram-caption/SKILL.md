# Instagramキャプション スキル v1.0

Instagram（@mynaturescape・@myraisingdiary）の投稿キャプションを作るときに発動する。
**目的**：creator/CLAUDE.md にルールを書いて毎回読む方式をやめ、IGキャプション作成タスクのときだけ本スキルを発動して全ルールを過不足なく適用する。

## 発動時に必ず行う（発動＝適用）

1. 対象アカウント（@mynaturescape か @myraisingdiary）と投稿種別（ショップ投稿／自宅植物／子育て）を確定する。
2. 下記の該当ルールを全て適用する。
3. 提出前に末尾の「自己チェック」を通してから PM レビューへ回す。

> このスキルは Instagram キャプション**専用**。note・ショップ記事・アフィリエイト文・Threadsのルールは含まない（各専用スキルを使う）。

## Instagram投稿時間ルール（@mynaturescape）

- **投稿時間：毎朝5:00**（2026-06-03 確定）
- 理由：フォロワーの主軸が40〜60代女性（早起き・朝のゆったり時間帯）。オーナーの体感（朝にいいねが来る）とデータが一致
- ブログ公開（22:00）の翌朝5:00に投稿するのが基本フロー

## Instagramキャプション ハッシュタグルール（ショップ投稿）

- **地名タグを必ず1つ入れる**：「花屋 亀戸」「植物ショップ 錦糸町」のように地名検索からの流入を狙う
- 構成例：`#カテゴリ #植物ジャンル #地名`（2〜3個）
- ハッシュタグブロック（大量羅列）は付けない

## Instagramキャプション フッタールール（ショップ投稿）

| ショップタイプ | フッター |
|---|---|
| YouTube動画あり | `YouTubeにて詳細を公開しています` |
| YouTube動画なし | `植物ブログにて詳細を公開しています` |

共通2行目以降：`リンクはプロフィールから` / `@mynaturescape`

## Instagram投稿（ショップ紹介カルーセル）必須ルール（2026-06-02 FB追記）

- **1枚目は必ず `Instagram用サムネイル.png`**: フォルダ内の他の写真を1枚目にしてはいけない
- **写真は極力多く使う**: 最大10枚まで使用する。フォルダ内に10枚未満なら全枚。10枚超えなら10枚選ぶ
- **構成順**: サムネイル(固定) → 外観 → 店頭・花苗 → 観葉植物 → 店内 の流れを基本とする

## Instagramキャプション / @mynaturescape（2026-06-03 全面改訂）

**基本方針：ファンを増やすための「読まれるキャプション」を書く**

```
【必須ルール】
□ 絵文字を使わない → シックで落ち着いた雰囲気を出す
□ フッターを付けない → ROOMリンク・SNS誘導・ハッシュタグブロック不要
□ 訴求・宣伝をしない → 売ろうとする言葉はゼロ
□ ハッシュタグは植物名など1〜2個のみ自然に入れるか、なし

【キャプションの質の基準】
□ 「刺さる言葉」があるか → 読んだ人が何かしら感情ポイントを持ち帰れるか
□ グッとくる・ハッとする・共感する、いずれかの感情が動くか
□ 自分の言葉で語られているか（自動生成的な均一文体はNG）
□ 一人称の体験・感情・観察が入っているか

【参考スタイル：@hare._room】
- 日常の観察を起点にする（「6月みを感じる」）
- 植物との関係性を語る（「定期的に剪定してるからか大きさはあまり変化なし」）
- 押し付けない・売らない・ただ語る
```

**Why:** 重いフッター・訴求キャプションはアカウントの温度を下げてファンが付かない。@hare._room（215いいね）、@tunagu1018（318いいね）はいずれも訴求なし。宣伝色がゼロのアカウントがエンゲージメントを獲得している（2026-06-03 オーナー確認）

**【キャプションスタイル：日記的一人称（2026-06-03 追加）】**

```
一般人のSNSは「日常」が商品。売るものがない分、体験・感情・気づきを語ることがコンテンツになる。

□ 「行った・感じた・思った」の一人称体験を必ず入れる
□ お店・植物の説明で終わらせない → 自分がどう感じたか・何が印象に残ったかを加える
□ 読者が「今日はどうだったんだろう」と思って戻ってきたくなる文章にする
□ 日常の感情・観察・驚きが「その人にしか書けない口コミ」になる
□ ショップ投稿・自宅植物投稿ともに適用する
```

**Why:** 定期的に届く日記的投稿がファン化を促す。店舗アカウントは商品が主役だが、個人アカウントは「その人自身」が主役。読者は日記を覗きたくなる心理を持つため、個人の感情・体験が入ったキャプションが共感・継続フォローにつながる（2026-06-03 オーナー確認）
