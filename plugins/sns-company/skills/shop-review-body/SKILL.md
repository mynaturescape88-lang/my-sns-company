# shop-review-body（本文の成果物レビュー）

下記の観点がすべて満たしていることを確認する。

- [ ] 写真ファイル名に無い植物名を断定せず汎用表現にしたか [[feedback_shop_article_rules]]
- [ ] 時間/季節/価格/電話/予約/内部管理（水揚げ等）を書いていないか [[feedback_shop_article_rules]]
- [ ] Web情報をそのまま引用せずオリジナルの言葉にしたか
- [ ] 見出しはh3のみ・h3直後に必ず本文pが続くか [[feedback_shop_article_rules]]
- [ ] 末尾に「まとめ」節（h3＋本文）があるか
- [ ] 記事内の文字数が2500字以上であること
- [ ] まとめを除く各見出し（セクション）が600字以上であること
- [ ] 体験記要素は固有名詞・実体験ベースで一般化していないか [[feedback_shop_article_rules]]
- [ ] 見出しはWeb引用のままでなく、興味を引く見出しとして生成し直したか
- [ ] 文末にリズムを持たせたか（です・ます調の単調な連続を避けたか）
- [ ] 作成した文章は適切な位置で改行されているか
- [ ] 「。」や話の転換があるにも関わらず、改行せず文章を続けて記載している箇所がないか
- [ ] 記載した文章と挿入した画像がリンクしているか
- [ ] 画像にないエリア・ゾーンを記載する場合、「写真にはないですが」等の補足文が入っているか

- **fail-closed**：1つでもfailなら提出せず→修正→観点確認へ戻る
- 全passのみ次工程（写真配置）へ進める
