# 章別素材台帳テンプレート（人智の外側 長尺・品質優先版）

> 工程3で作る**制作の背骨**。全カットを1枚に並べ「画・区分・尺・予cr/実cr/済」で管理する。
> 実証済の正本例＝`drafts/nazca-cr-ledger.md`（第1弾ナスカ・全カット✅）。新題材は本テンプレを複製して `drafts/<topic>-cr-ledger.md` に作る。
> この台帳が**単一の正本**（task_log のインライン表より優先）。

## 使い方（手順）
1. 工程2の台本が確定したら、**台本の各文/章を「再生順マスターショットリスト」に分解**する（A表）。
2. 各カットに**区分①〜⑥／★／(間)** を割り当て、**画（何を映すか）** と **尺** を入れる。
3. **予cr**を積算（区分④⑤⑥のみcr・他は0）→ **1本上限300cr** と照合（B/Dサマリ）。超過時のみ最小劣化で削る。
4. 素材ができたら **済✅** を立て、AI動画は **実cr** を記入（0にしない・失敗再生成は全試行合計）。
5. 章ごとの **想定尺（実測ナレ長）＝割付尺** を C表で突き合わせ、差分ゼロに敷き詰める。

## 区分の凡例（必須・冒頭に明記する）
| 区分 | 中身 | cr |
|---|---|---|
| ① | 実写無料素材（Pexels等・H.264選別） | 0 |
| ② | Web UI 0cr画像（自作静止画・Ken Burns） | 0 |
| ③ | 図解（PIL内製・暗背景/金/W6・**2連続禁止**・下240px字幕帯） | 0 |
| ④ | 語り部口パク動画（**Seedance 2.0 std720p**・4.5cr/s・最小4s） | cr |
| ⑤ | 無音モーション動画（**Kling v3.0 std720p sound off**・7.5cr/カット） | cr |
| ⑥ | AI再現カット動画（同Kling・7.5cr/カット） | cr |
| ★ | 0cr合成（numpyグロー×ffmpeg blend=screen） | 0 |
| (間) | 章境・語り部の間（黒みフェード/無音） | 0 |

> 済凡例：✅＝済 or 不要（生成済AI/実写stock/間） ／ ⬜＝未着手（図解・★合成・AI画像・AI動画でまだ作っていない）

### 第1弾ナスカでの確定値（参考・新題材の見積り基準）
- 単価＝語り部口パク④＝Seedance std720p **4.5cr/s**（最小4s）／⑤⑥＝Kling std720p sound off **7.5cr/カット**（5s基準）。
- クリップは5s基準で生成し、ビート尺はKen Burns/スロー/隣接0crカットで充足（一方向モーションはスロー/ループ不可）。
- 実消費の目安＝1本 約170cr（語り部口パク5＋stinger＋再現4＋既存B-roll5・残りは全て0cr素材）。
- 教訓＝**口パクは1テイクで通す前提でcr見積り**（ナスカ中盤は2回作り直しで+45cr超過）。詳細＝`lessons-learned.md`。
