# SNS戦略・分析ガイドライン

## SNSマーケター・アナリスト共通の考え方

### 目標設定：SMART法則

目標はSMART法則に従って設定する。

| 要素 | 意味 | 例 |
|---|---|---|
| **Specific（具体的）** | 何を達成するか明確に | 「SNSを頑張る」→「Instagramフォロワーを増やす」 |
| **Measurable（測定可能）** | 数値で測れる | 「フォロワー1,000人達成」 |
| **Achievable（達成可能）** | 現実的な目標 | 現状100人→3ヶ月で1,000人 |
| **Related（関連性）** | ビジネス目標と紐づく | フォロワー増→問い合わせ増→売上増 |
| **Time-bound（期限）** | いつまでに | 「3ヶ月以内に」 |

### 目標設定の構造

```
KGI（最終目標 ＝ ビジネス目標に直結）
  └── KPI（中間指標 ＝ KGI達成のための測定値）
        └── 施策（具体的なアクション）
```

**KPIの3段階:**

| 目的 | 指標 |
|---|---|
| **認知拡大** | インプレッション数、リーチ数、フォロワー増加率 |
| **エンゲージメント向上** | エンゲージメント率 ＝（いいね＋コメント＋シェア＋クリック）÷ インプレッション数 |
| **コンバージョン** | CTR（クリック率）、CVR（転換率）、SNS経由ROI |

**例:**
- KGI: 月10件の問い合わせ獲得
- KPI: プロフィールリンククリック数 月100回 / エンゲージメント率 3%以上
- 施策: 週3回投稿 / ストーリーズ毎日更新 / 投稿に必ずCTA（行動喚起）を入れる

## 戦略立案ガイドライン（SNSマーケター用）

### 戦略立案の基本ステップ

1. **目標設定**: SMART法則でKGI・KPIを定義する
2. **ターゲット設定（ペルソナ）**: 性別・年齢・地域・興味関心・オンライン行動・内面的動機まで具体化する
3. **現状把握（SNS監査）**: 既存アカウントの数値・コンテンツ・競合を確認する
4. **競合分析**: 参考アカウントのコンテンツ・指標をベンチマークとして記録する
5. **プラットフォーム選定**: ターゲット層がいる場所を選ぶ（複数可）
6. **コンテンツ戦略設計**: 投稿タイプ・頻度・トーンを定める
7. **施策の優先順位**: インパクト×実行難易度で優先度をつける
8. **スケジュール化**: いつ・何を・どう実施するかを具体化する

### コンテンツ戦略の5類型

| 類型 | 内容 | 向いている目的 |
|---|---|---|
| **データ駆動型トレンド投稿** | トレンド・ニュースに乗って発信 | 認知拡大・リーチ増 |
| **コミュニティ形成** | 対話・質問・アンケート・ライブ配信 | エンゲージメント・信頼構築 |
| **UGC（ユーザー生成コンテンツ）** | フォロワーの投稿をシェア・リポスト | 信頼性向上・拡散 |
| **インフルエンサーマーケティング** | 影響力ある人物との協力 | 認知拡大・ターゲット層へのリーチ |
| **商品・サービス紹介** | CTA付きの直接訴求 | コンバージョン |

### 投稿カレンダー設計

| 種別 | 割合（目安） | 目的 |
|---|---|---|
| **情報提供・有益コンテンツ（教育系）** | 60% | フォロワーの役に立つ。保存・シェアを促す |
| **ブランド・人柄紹介** | 20% | 信頼・親近感の醸成 |
| **CTA（行動喚起・プロモーション系）** | 20% | リンククリック・問い合わせ・購買への誘導 |

### プラットフォーム別の特性

| プラットフォーム | ユーザー層 | 重視する指標 | コンテンツ特性 | 推奨頻度 |
|---|---|---|---|---|
| **Instagram** | 幅広い（特に20〜40代） | 保存数・リーチ数・フォロワー数 | ビジュアル重視。統一感・世界観が重要。ショッピング機能あり | 週3〜5回（フィード）/ 毎日（ストーリーズ） |
| **X（Twitter）** | 幅広い（情報感度高め） | インプレッション・エンゲージメント | テキスト中心。リアルタイム性・共感性が重要 | 1日1〜5回 |
| **TikTok** | 10〜20代中心 | 再生数・フォロワー数・いいね数 | 動画。最初の1〜3秒が勝負 | 1日1〜数回 |
| **YouTube** | 全年齢層 | 視聴回数・登録者数・視聴維持率 | 長尺動画。タイトル・サムネイルが重要 | 週1〜2回 |
| **Facebook** | 30〜40代中心 | リーチ数・シェア数 | コミュニティ形成・拡散に強い。実名制 | 1日1〜2回 |

### エンゲージメント向上のテクニック

- 質問を投げかけて返信を促す
- アンケート・投票機能を活用する
- ライブ配信でリアルタイム交流する
- フォロワーの投稿・コメントに積極的に反応する
- オリジナルハッシュタグを育ててコミュニティを形成する
- トレンドに素早く乗る（ただし無理に乗らない）

### 戦略立案時のチェックリスト

- [ ] SMART法則でKGI/KPIを設定したか
- [ ] ターゲット層のペルソナを具体的に定義したか
- [ ] 競合・参考アカウントのベンチマークを確認したか
- [ ] プラットフォームの特性とターゲット層が一致しているか
- [ ] 実施可能なリソース（時間・コスト）を考慮しているか
- [ ] 有料広告・課金サービスを伴う提案は「要ユーザー確認」と明記したか

## 分析ガイドライン（アナリスト用）

### 分析の目的

SNS分析の主な目的：

1. **消費者インサイトの収集**: ユーザーの本音・ニーズを匿名データから把握する
2. **ブランド認知の追跡**: 自社がどう語られているかを監視する
3. **施策効果の測定**: 投稿・キャンペーンのROIを可視化する
4. **市場トレンド予測**: 競合や業界動向を先読みする
5. **危機管理**: 炎上・ネガティブ投稿の早期検知

### 主要指標一覧

| 指標 | 説明 | 改善のヒント |
|---|---|---|
| **フォロワー数** | アカウントを追っている人数 | 増加率・純増数を見る |
| **インプレッション** | 投稿が表示された回数 | ハッシュタグ・投稿時間で改善 |
| **リーチ数** | 投稿を見たユニーク人数 | インプレッション÷リーチで露出効率を確認 |
| **エンゲージメント数** | いいね・コメント・保存・シェアの合計 | 内容の共感度を示す |
| **エンゲージメント率** | エンゲージメント数÷リーチ数×100 | 3〜5%が目安（プラットフォームによる） |
| **保存数（Instagram）** | 投稿を保存した数 | 有益コンテンツの指標。アルゴリズムへの影響大 |
| **プロフィールアクセス** | プロフィールへの流入数 | 興味を持った人数の指標 |
| **リンククリック数** | プロフィールリンクのクリック数 | ビジネス系KGIへの直結指標 |
| **CTR（クリック率）** | クリック数÷インプレッション数×100 | コンテンツの訴求力を測る |
| **CVR（転換率）** | コンバージョン数÷クリック数×100 | 問い合わせ・購買への転換効率 |

### 分析手法

| 手法 | 内容 |
|---|---|
| **テキスト分析** | 投稿・コメントの感情・ニュアンスをAIで分析 |
| **感情分析（センチメント分析）** | ポジティブ / ネガティブ / ニュートラルに分類。ブランドイメージの把握に使う |
| **SOV分析（シェア・オブ・ボイス）** | 自社への言及が業界全体の何%を占めるかを測定 |
| **A/Bテスト** | キャプション文面・画像構成・CTA文言・投稿時間帯を変えて効果を比較 |

### リアルタイム対応型PDCA

教科書通りのPDCAより「今何が起きているか」に素早く対応することが重要。

```
Plan（計画）→ Do（実施）→ Check（即時確認）→ Act（素早い軌道修正）
```

- 小単位での即座な修正を繰り返す
- アルゴリズム変化・トレンドには柔軟に対応する
- KPIは定期的に見直し、目的に合わなくなったら更新する

### 分析レポートの構成

1. **サマリ（3行）**: 期間・主要指標・総括コメント
2. **目標vs実績**: KPIごとの比較表
3. **良かった投稿TOP3**: 数値の根拠と理由の仮説
4. **振るわなかった投稿**: 原因仮説と学び
5. **改善提案**: 次の期間に実施すること（具体的なアクション2〜3点）
6. **次回確認指標**: 次回レポートで重点確認する数値

### 分析時の注意点

- 数値はユーザーが提供したものを使用する（自己推測しない）
- 「なぜそうなったか」の仮説を必ず添える（数値の羅列にしない）
- 改善提案は「具体的なアクション」まで落とし込む（「投稿を増やす」ではなく「週3回→週5回、曜日を固定する」）
- データが完全に正確とは限らない点を認識する（スパム・誤情報・複数アカウントの影響がある）
- 外部要因（アルゴリズム変更・季節性・トレンド）も考慮する
- 事業貢献の根拠への翻訳を意識する（エンゲージメント数だけでなく「この数値がどうビジネスに影響するか」を説明する）

## 失敗パターン・注意事項

| 失敗パターン | 対策 |
|---|---|
| プラットフォーム特性を無視した戦略 | Facebookで有効な手法がTikTokでは効果なし。必ず各プラットフォームの特性を理解する |
| 戦略なく投稿を続ける | 目的・KPIなしの運用はリソース浪費。最初に目標設定を必ず行う |
| 投稿数増加による質の低下 | 量より質。リソース管理が重要 |
| ネガティブ情報の軽視 | 炎上は一気に拡散する。ネガティブな反応は早期に把握・対処する |
| 短期で結果を求める | SNS効果が出るまでに3〜6ヶ月かかるのが一般的。継続が最重要 |
