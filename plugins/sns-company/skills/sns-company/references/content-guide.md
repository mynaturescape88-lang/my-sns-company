# コンテンツ制作ガイドライン

## コンテンツ制作の基本原則

1. **目的を持って作る**: 「何のために投稿するか」を常に意識する。目的なき投稿は継続できない
2. **ターゲット層を常に意識する**: 「この人はなぜこの投稿を読むのか」を考える
3. **トーン&マナーを統一する**: アカウントの世界観・言葉遣いを守る
4. **CTA（行動喚起）を入れる**: 読んだ後に何をしてほしいかを明確にする
5. **最初の一文・冒頭数秒で掴む**: スクロールを止める冒頭が最重要
6. **継続を最優先にする**: SNSは「続けた人が勝つ」世界。完璧より継続

---

## 投稿タイプの4分類

すべての投稿をこの4分類に意識的に当てはめて、バランスよく組み合わせる。

| タイプ | 目的 | 特徴 |
|---|---|---|
| **エンタメ系** | 拡散・認知拡大 | 笑い・驚き・感情を動かす。シェアされやすい |
| **教育系（お役立ち）** | フォロワー増加・保存獲得 | 「使えるノウハウ」「知らなかった情報」。保存・シェアにつながる |
| **プロモーション系** | 問い合わせ・売上化 | 商品・サービスの紹介。CTA必須 |
| **コミュニケーション系** | 信頼構築・エンゲージメント | 質問・アンケート・返信。フォロワーとの対話 |

**目安の比率:** エンタメ・教育系 60% / コミュニケーション系 20% / プロモーション系 20%

## コンテンツ種別と制作ポイント

### テキスト投稿（X / Facebook / Threads）

| 要素 | ポイント |
|---|---|
| **冒頭** | 最初の1〜2文で読む価値を伝える。「〇〇な人に読んでほしい」「実は〇〇だった」など |
| **構成** | 結論→理由→具体例→CTA の順が読みやすい |
| **長さ** | X: 140〜280文字が目安 / Facebook・Threads: 適切な長さで |
| **改行** | 詰め込みすぎない。視覚的に読みやすい行間を意識する |

### キャプション（Instagram / TikTok）

| 要素 | ポイント |
|---|---|
| **冒頭** | 「もっと見る」より前の1〜2行が勝負 |
| **本文** | 画像・動画の補足情報や文脈を伝える |
| **ハッシュタグ** | 本文と分けて最後に配置するのが一般的 |
| **CTA** | 「保存してね」「コメントで教えて」など具体的な行動を促す |

### 動画コンテンツ（TikTok / Instagram リール / YouTube Shorts）

| 要素 | ポイント |
|---|---|
| **冒頭1〜3秒** | 視聴者が離脱するかを決める最重要ポイント。問いかけ・驚きで引き込む |
| **テキストオーバーレイ** | 音声オフでも内容が伝わるようにする |
| **尺** | TikTok/リール: 15〜60秒が目安 / YouTube Shorts: 60秒以内 |
| **CTA** | 動画内または概要欄でアクションを促す |
| **視覚要素** | 画像・動画はテキストより高反応。クオリティより「最初の引き」が重要 |

### ハッシュタグの選定基準

| 種別 | 投稿数目安 | 目的 |
|---|---|---|
| **ビッグタグ** | 100万件以上 | リーチ拡大（埋もれやすい） |
| **ミドルタグ** | 1万〜100万件 | バランス型（推奨） |
| **スモールタグ** | 1万件未満 | ニッチ層への訴求 |

- 投稿テーマに関連するタグを選ぶ（数だけ詰め込まない）
- アカウント固有のブランドタグを1つ持つと認知が積み上がる
- ハッシュタグ活用でエンゲージメントが大きく伸びる事例あり

## 制作フロー

```
1. SNSマーケターから方針・テーマを受け取る（戦略・KPIとの整合確認）
   ↓
2. ターゲット層・トーン&マナー・投稿タイプを確認する
   ↓
3. コンテンツ案を作成する（複数案出しが望ましい）
   ↓
4. 品質チェックリストでセルフチェックする
   ↓
5. PMにレビュー依頼する
   ↓
6. PM承認後、秘書を通じてユーザーへ提示する（必須）
```

## 効率的な制作のコツ

- **ストック制作**: 1日に複数本まとめて作成し、予約投稿機能で配信する。ネタ切れ防止と安定配信を実現
- **初期は週3回から**: 無理のないペースで始めて習慣化する
- **最適時間帯の検証**: 投稿後のデータからエンゲージメントが高い時間帯を特定する
- **A/Bテストの実施**: キャプション文面・CTA・投稿時間帯を変えて効果を比較する
- **フォロワーとの交流**: 初期段階でコメント返信を重視すると信頼構築が加速する

## 品質チェックリスト

制作後、PMへのレビュー依頼前に必ず確認する。

- [ ] ターゲット層が共感・有益だと感じる内容か
- [ ] アカウントのトーン&マナーと一致しているか
- [ ] 冒頭で読む価値が伝わるか
- [ ] 誤字・脱字・文法ミスがないか
- [ ] CTAが明確か
- [ ] ハッシュタグは内容と関連しているか
- [ ] 事前にファクトチェック（事実確認）を行ったか
- [ ] 著作権・肖像権のリスクがないか
- [ ] 薬機法・景品表示法に違反する表現がないか
- [ ] 炎上リスクのある表現・差別的な表現がないか
- [ ] ステルスマーケティングに該当する表現がないか
- [ ] 個人情報・プライバシーへの配慮がされているか

## NGワード・表現・法的リスク

| カテゴリ | NG例 | 理由 |
|---|---|---|
| 誇大広告 | 「絶対に痩せる」「100%成功する」 | 景品表示法・薬機法違反リスク |
| ステマ | 広告表記なしのPR投稿 | 景品表示法違反（2023年10月より厳格化） |
| 差別・ヘイト | 特定属性を否定・侮辱する表現 | 炎上・アカウント停止リスク |
| 著作権 | 他者の画像・文章・音楽の無断転載 | 著作権法違反リスク |
| 肖像権 | 無断での他者の顔出し | 肖像権・プライバシー侵害 |
| 個人情報 | フォロワーのDM内容・個人情報の公開 | 個人情報保護法違反リスク |
| 不正確な情報 | ファクトチェックなしの情報拡散 | 信頼失墜・炎上リスク |

## 炎上対策

- **投稿前のダブルチェック**: 送信前に必ず見直す
- **ファクトチェック**: 事実情報を含む投稿は根拠を確認する
- **センシティブトピックは慎重に**: 政治・宗教・差別に関わるテーマは原則扱わない
- **対処マニュアルを持つ**: 万が一の炎上時にどう対処するかを事前にユーザーと確認しておく
- **ネガティブコメントへの対応**: 感情的な反論はせず、冷静かつ誠実に対応する
