# my-sns-company

Claude Codeで動く仮想SNS運営組織プラグイン。
秘書・PM・SNSマーケター・アナリスト・コンテンツ制作・SEが役割分担してSNS運営を進める。

## インストール

```
/plugin marketplace add mynaturescape88-lang/my-sns-company
/plugin install sns-company@my-sns-company
```

## 使い方

プロジェクトディレクトリで実行：

```
/sns-company
```

初回は秘書がヒアリングを行い、`.company/` を自動生成する。
2回目以降は秘書が窓口としてSNS運営を進める。

## 組織構成

| 役割 | 責務 |
|---|---|
| 秘書 | ユーザーの唯一の窓口。進捗報告・質問対応・振り分け |
| PM | 全体管理・成果物レビュー・戦略承認 |
| SNSマーケター | 目標達成のための戦略立案・施策設計 |
| アナリスト | 数値分析・現状把握・改善提案 |
| コンテンツ制作 | 投稿記事・キャプション・コンテンツ作成 |
| SE | 運営に必要なツール・自動化の設計・作成 |

## 主な用途

- 投稿コンテンツの作成・企画
- SNS数値の分析とレポート
- 運営戦略の立案
- 運営ツールの開発

## ガイドライン

- セキュリティガイドライン（APIキー・認証情報管理）
- SNS戦略ガイドライン（KPI設定・分析観点）
- コンテンツ制作ガイドライン（品質基準・作成フロー）
